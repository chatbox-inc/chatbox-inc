### フォルダ構成

````
png/
	square_light/ 正方形 基本色アイコン
	square_dark/  正方形 白アイコン
	banner_light/ 長方形 基本色アイコン
	banner_dark/  正方形 白アイコン
svg/ 
	square_light/
	square_dark/
	banner_light/ 
	banner_dark/ 
````

- luqid : 背景透過データ
- white : 背景白マット付きデータ
